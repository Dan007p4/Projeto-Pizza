Um projeto realizando todo o trabalho de data science em um banco de dados do Keggle sobre o histórico de vendas de uma pizzaria durante um ano.

Foi realizada a limpeza e análise dos dados ademais a elaboração de um modelo preditivo sobre os dados escolhidos.

O objetivo do modelo é designar a melhor pizza a ser produzida em cada momento com base nas informações do dia, horário e etc. 

Dados obtidos em:
https://www.kaggle.com/datasets/shilongzhuang/pizza-sales
